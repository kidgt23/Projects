Project 1: Automatic Fan

Using Matlab along with an Arduino I created a project where a Fan would automatically turn on and off depending on the temperature and moisture in the room. This was done using a temperature sensor that was attached to the Arduino and a moisture sensor attached using extra pin set on the board. The system also had a small LED screen along with a button schematic to help start the system. The values were graphed on the screen as temperature versus time and moisture versus time. The moisture values could be seen either on the graph or on the board itself. The developer can also just view these values in the command prompt of the Arduino or Matlab.

Project 2: 2D Parking Booking System GUI

Development and Implementation of a Comprehensive Parking Booking System: Spearheaded the development of a GUI-based Java application for efficient parking management, catering to diverse user groups including students, faculty, non-faculty staff, and visitors. The system features robust user authentication, dynamic rate calculation based on user type, and sensor integration for real-time parking space monitoring. Implemented features like automatic account generation for management, user-specific parking rates, and sensor-based parking space detection and reservation. Advanced Feature Integration and System Management: Contributed to the system’s extensive functionality, enabling features such as parking space management by system administrators, unique identification for each parking space, and user-friendly navigation aids. Facilitated user interactions with the system through functionalities like booking editing and cancellation, parking time extension, and diverse payment options, enhancing user experience and operational efficiency. Rigorous Testing Using EvoSuite and Randoop: Ensured the reliability and robustness of the system by employing advanced automated testing tools, EvoSuite and Randoop. Utilized EvoSuite for generating high-coverage test suites focusing on the system's critical functionalities, and Randoop for regression testing to identify any unintended changes or errors in the system’s behavior. This strategic approach to testing played a pivotal role in maintaining high software quality and enhancing system reliability.
